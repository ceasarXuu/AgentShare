# NBAgents - Multi-Agent System

该项目托管了多个专用的 AI Agent。目前的 Agent 列表如下：

## Available Agents

### 1. [NBResearcher](agents/NBResearcher/README.md)
**商业机会研究员**
面向独立开发者与小团队，目标是在有限时间内快速找到当下最值得做的 App/软件产品方向，并把机会从“想法”推进到“可验证的方案”。

---

## 目录结构

- `agents/`: 存放所有 Agent 的目录
  - `NBResearcher/`: 商业机会研究员 Agent
- `.opencode/`: OpenCode 配置文件
  - `agents/`: Agent 定义文件

## 快速开始

### 使用 NBResearcher
1. 确保已安装 OpenCode。
2. 配置 `.env` 文件（参考 `agents/NBResearcher/.env.example`）。
3. 选择 NBResearcher Agent 并开始对话。
