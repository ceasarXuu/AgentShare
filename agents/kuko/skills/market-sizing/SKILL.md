# 市场规模测算

## Overview
用自下而上模型估算市场规模，并用场景规划验证合理性。

## Workflow
1. 明确目标人群与定价假设
2. 计算TAM并给出数据依据
3. 计算SAM并解释过滤条件
4. 计算SOM并基于转化路径
5. 做乐观/基础/悲观场景对比

## TAM/SAM/SOM
- TAM: 全量可用市场
- SAM: 可服务市场(人群/地域/平台过滤)
- SOM: 可获得市场(转化路径与执行约束)

## 场景规划
- 乐观/基础/悲观: 明确增长、转化与收入假设
- 输出对应策略或阈值

## Output
- TAM/SAM/SOM表格与假设来源
- 场景规划对比表

## Guardrails
- 只用真实数据和可执行路径
- 不用“0.1%市场份额”式推导
