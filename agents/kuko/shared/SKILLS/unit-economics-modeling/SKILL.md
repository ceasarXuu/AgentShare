# 单位经济模型

## Overview
用单位经济指标判断商业模式是否可持续。

## Workflow
1. 明确获客成本、定价、毛利与留存假设
2. 计算CAC并列出来源
3. 计算LTV并拆分ARPU与留存
4. 计算LTV/CAC与回本周期
5. 做关键参数敏感度分析

## 指标定义
- CAC: 获客成本
- LTV: 客户终身价值
- Payback Period: 回本周期
- Churn Rate: 流失率

## Output
- 单位经济表(CAC/LTV/回本周期)
- 关键参数敏感度分析
- 健康度结论与改进方向

## Guardrails
- 假设必须可追溯
- LTV/CAC < 3 需标记风险
