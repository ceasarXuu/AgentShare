# 竞品与Gap分析

## Overview
用竞品分层+矩阵打分发现可突破的价值缺口。

## Workflow
1. 划分竞品四层: 直接/间接/替代/潜在
2. 建立功能-定价-体验矩阵并打分
3. 识别用户需求与竞品覆盖的Gap
4. 评估竞争强度(可选五力)
5. 输出差异化策略与避免区

## 矩阵要素
- 功能: 核心能力、关键特性
- 定价: 订阅/买断/广告/增值
- 体验: 界面、学习成本、情感共鸣

## Gap识别
- 需求与现有解决方案逐项比对
- 标注“巨大/中等/小”缺口

## Porter五力(可选)
- 现有竞争者、新进入者、替代品、买方议价、供应商议价

## Output
- 竞品矩阵与打分表
- 功能Gap清单与机会排序
- 差异化策略与避险建议

## Guardrails
- 竞品覆盖必须包含替代方案
- 避免“无竞争”结论
