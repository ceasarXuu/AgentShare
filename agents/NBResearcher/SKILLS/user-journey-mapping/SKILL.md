# 用户旅程映射

## Overview
通过阶段-行为-情绪映射找到真正需要被解决的“心理节点”。

## Workflow
1. 定义用户画像与触发场景
2. 划分旅程阶段并记录行为
3. 标注每阶段情绪与问题
4. 标记可介入的关键节点
5. 产出产品设计洞察与介入策略

## 旅程要素
- 阶段: 触发 → 察觉 → 焦虑升级 → 强迫行为 → 求助 → 解决
- 维度: 行为/情绪/痛点/现有替代方案

## Output
- 用户旅程图(阶段/情绪/行为/痛点)
- 关键介入点与产品策略

## Guardrails
- 旅程来自真实用户原话与行为
- 介入点要能对应可实现的功能
