# 模板二：命题研究型报告（Directed Research Report）

**适用场景**
- 给了研究方向 / 问题

**核心目标**
- 对该方向进行系统性拆解、验证与商业判断

# 《商业机会命题研究报告》

## 0. Executive Summary（执行摘要）

### 研究问题
- 本报告围绕【你给定的研究命题】展开系统分析。

### 核心结论
- 在当前条件下，该方向表现为【可行 / 条件可行 / 不具备商业价值】。

### 关键依据
- 市场与需求层面
- 用户行为与付费意愿
- 商业模型与实现成本

## 1. 研究背景与问题定义

### 1.1 背景与研究动因
- 行业 / 平台 / 技术背景
- 为什么该问题值得被研究
- 当前讨论或实践中的模糊点

### 1.2 明确研究问题与边界

清晰定义：
- 研究对象
- 不包含的内容
- 研究假设（如有）

## 2. 用户与需求分析

### 2.1 目标用户与使用场景
- 用户角色
- 关键触发场景
- 期望达成的目标

### 2.2 用户行为与需求证据
- 搜索行为
- 使用频率
- 用户讨论与反馈

## 3. 市场与竞品结构分析

### 3.1 现有解决方案概览
- 产品类型
- 功能分布
- 定价模型

### 3.2 结构性问题与差异点
- 未被满足的需求
- 过度设计或错配部分

## 4. 商业化与可行性分析

### 4.1 商业模式拆解
- 潜在收费方式
- 收费逻辑与用户心理

### 4.2 成本与实现复杂度评估
- 技术复杂度
- 长期维护成本
- 个人开发可行性

## 5. 风险与不确定性分析（允许）
- 平台政策风险
- 需求可持续性
- 竞争与替代风险

## 6. 结论与建议
- 是否值得投入
- 推荐的下一步动作
- MVP
- 深度验证
- 放弃或延后

## 7. 附录：证据与数据来源
- 原始数据链接
- 用户原话
- 截图或表格
